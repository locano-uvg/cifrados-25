Cifrados Simétricos

Agenda

  • Definición
  • Funcionamiento
  • Eficiencia
  • Clasificación
  • Algoritmos Populares

Definición

  • Utilizan la misma clave para cifrar y descifrar
  • Clave compartida entre emisor y receptor
  • Clave debe ser segura

Funcionamiento

  • Texto plano (m)
  • Clave (k)
  • Texto cifrado (c)

c = E(m, k)

Eficiencia

  • Rápidos
  • Claves cortas
  • Algoritmos eficientes
  • Son generalmente más eficientes en términos de velocidad de procesamiento en comparación con los cifrados asimétricos (de clave pública).
  • Esto los hace ideales para cifrar grandes cantidades de datos.

Clasificación

  • Flujo (Stream)
  • Bloque (Block)

Flujo (Stream)

  • Procesan los datos bit a bit
  • Generan un flujo de bits cifrados
  • Se combinan con el texto plano mediante XOR
  • Se utilizan en comunicaciones en tiempo real

Bloque (Block)

  • Procesan los datos en bloques
  • Dividen el texto plano en bloques
  • Cada bloque se cifra de manera independiente utilizando la misma clave
  • Se utilizan en almacenamiento de datos

Algoritmos Populares

Stream Ciphers

Algoritmo Uso
OTP (One-Time Pad) Comunicaciones militares, diplomáticas y gubernamentales
RC4 (Rivest Cipher 4) (Obsoleto) WEP (Wired Equivalent Privacy)
ChaCha20 TLS 1.3, WireGuard, Signal, WhatsApp, OpenSSH, Cloudflare

Stream Ciphers

Algoritmo Uso
Salsa20 cifrado de disco y seguridad en aplicaciones
Grain Sensores, IoT, comunicación de dispositivos de bajo consumo
Trivium Dispositivos IoT, smart cards y sistemas embebidos

Block Ciphers

Algoritmo Uso

AES (Advanced Encryption Standard) Transacciones financieras, comunicaciones seguras
