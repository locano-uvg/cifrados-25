Stream Ciphers

Agenda

  • Definición
  • Características
  • Keystreams
  • Procesos

Definición

  • Utilizan una clave y un flujo de bits
  • Clave compartida entre emisor y receptor
  • Clave debe ser segura
  • Cifra cada bit de manera independiente
  • Cifra los datos a medida que fluyen

Características

  • Velocidad
  • Facilidad de implementación
  • Cifrado en tiempo real
    • Ideal para transmisiones de datos, como videollamadas o cifrado de redes inalámbricas

Características

  • Funciona generando una secuencia de claves (keystream) que se combina con el texto plano mediante una operación XOR
  • Claves cortas
  • Sensibles a la reutilización de claves
    • Si se usa la misma clave para cifrar múltiples mensajes, un atacante puede deducir información.

Keystreams

  • Secuencia de bits generada por el algoritmo a partir de una clave y nonce
  • Se combina con el texto plano mediante XOR
  • Se utiliza una vez y se descarta
  • Depende de la clave y el vector de inicialización

Keystreams

Es una cadena de datos infinita que contiene caracteres aleatorios o pseudoaleatorios, los cuales se utilizan como clave para encriptar datos y mantenerlos en secreto.

Nonce

Es un valor numérico que se utiliza una vez y se descarta. Se utiliza para evitar la reutilización de claves.

Longitud de la clave

La longitud de la clave es un factor importante en la seguridad del cifrado. Una clave más larga es más segura.

Se refiere al número de bits que componen la clave utilizada por el algoritmo de cifrado

Longitud de la clave

Longitud de la clave Seguridad
64 bits Baja
128 bits Media
256 bits Alta

Proceso

  • Generación de keystream
  • Operación XOR
  • Descifrado

Generación de keystream:

Se genera una secuencia de bits aleatorios o pseudoaleatorios a partir de la clave y el vector de inicialización.

Flujo Pseudoaleatorio

  • Utilizan un flujo pseudoaleatorio de bits que se genera mediante un algoritmo de generación de números pseudoaleatorios (PRNG).
  • La clave y, en algunos casos, un vector de inicialización (IV) alimentan el PRNG para generar la secuencia pseudoaleatoria

Operación XOR:

Se combina el keystream con el texto plano mediante una operación XOR.

Descifrado:

Se realiza el proceso inverso para obtener el texto plano.
