Funciones Hash

¿Qué es una Función Hash?

  • Una función hash es un algoritmo que convierte una entrada de cualquier tamaño en una salida de longitud fija.
  • Esta transformación se realiza mediante un algoritmo de hash específico, que aplica una serie de operaciones matemáticas para producir el resultado final.
  • Utilizada para el almacenamiento de contraseñas y verificación de integridad.

Propiedades de una Función Hash

  • Unidireccionalidad:
    Es fácil calcular el hash de un mensaje, pero es computacionalmente difícil o imposible recuperar el mensaje original a partir del hash.
  • Determinismo:
    La misma entrada siempre produce el mismo hash.
  • Uniformidad:
    Pequeños cambios en la entrada deben producir cambios significativos en el hash.
  • Resistencia a colisiones:
    Dos entradas distintas no deben producir el mismo hash.
  • Rapidez:
    Cálculo eficiente incluso con grandes volúmenes de datos.

Algoritmos Hash Modernos

MD5

  • Desarrollado en 1991.
  • Produce un valor de hash de 128-bit
  • Considerado inseguro por vulnerabilidades.
  • Usado para verificar integridad de archivos.

SHA

SHA significa (Secure Hash Algorithm) Algoritmo de Hash Seguro

SHA-1

  • Desarrollado en 1995.
  • Produce un valor de hash de 160-bit.
  • Produce un valor hexadecimal de 40 dígitos
  • Considerado inseguro por vulnerabilidades.
  • Usado en firmas digitales y autenticación.

SHA-256

  • Desarrollado en 2001.
  • Produce un valor de hash de 256-bit.
  • Produce un valor hexadecimal de 64 dígitos
  • Ampliamente utilizado en aplicaciones de seguridad.
  • Usado en Blockchain y protección de contraseñas.

SHA-3

  • Desarrollado en 2015.
  • Produce un valor de hash de 224, 256, 384 o 512-bit.
  • Considerado más seguro que SHA-2.
  • Usado en aplicaciones críticas de seguridad.

BLAKE3

  • Desarrollado en 2020.
  • Produce un valor de hash de 256-bit.
  • Considerado más rápido y seguro que SHA-3.
  • Usado en aplicaciones de alto rendimiento.

Argon2

  • Desarrollado en 2015.
  • Función hash específica para proteger contraseñas.
  • Considerado más seguro que otros algoritmos.
  • Usado en aplicaciones de autenticación.

Aplicaciones

  • Verificación de Integridad de Datos:
    Verificar que un conjunto de datos no ha sido alterado o corrompido.
  • Protección de Contraseñas:
    Almacenar contraseñas de forma segura mediante el almacenamiento del hash en lugar de la contraseña en sí misma.
  • Autenticación de Mensajes :
    Generar firmas digitales para autenticar la fuente y la integridad de un mensaje.
  • Indexación y Búsqueda Eficiente :
    Generar firmas digitales para autenticar la fuente y la integridad de un mensaje.

Blockchain

Bitcoin utiliza SHA-256 para generar bloques de transacciones.

Firmas Digitales

SHA-1 y SHA-256 son usados para autenticar mensajes y documentos.

Protección de Contraseñas

Argon2 es usado para proteger contraseñas en bases de datos.

Verificación de Integridad

SHA-256 es usado para verificar la integridad de archivos y datos.

Hash en validaciones

Un uso típico de las funciones hash es hacer verificaciones de validaciones.

Un uso frecuente es la validación de colecciones comprimidas de archivos, tales como archivos .zip o .tar

Dado un archivo y su valor de hash esperado puedes hacer tu propio cálculo de hash para validar que el archivo que recibiste está completo y no corrupto.

Ejemplos de Seguridad

Las empresas de antivirus como Kaspersky Lab, utilizan las funciones criptográficas hash para detectar malware.

Las compañías discográficas y cinematográficas crean listas de hash rechazados para proteger sus datos. Existe también listas de hash de malware y muchas de ellas son de acceso público.

Estas listas, o firmas de malware (como también se las conoce), están conformadas por valores hash de las piezas de malware o de algunas partes pequeñas y reconocibles de estas piezas.

Por un lado, si un usuario detecta una archivo sospechoso, puede consultar en las bases de datos públicas y, de esta forma, sabrá si se trata o no de un archivo malicioso.

Por el otro, los productos antivirus detectan y bloquean los malware comparando los hash con sus propias bases de datos y con las públicas.

Implementación

Ejemplo en Python: SHA-256


      import hashlib
      mensaje = "Hola, mundo"
      hash_sha256 = hashlib.sha256(mensaje.encode()).hexdigest()
      print("SHA-256:", hash_sha256)

Uso de Hash en Contraseñas (Argon2)


      from argon2 import PasswordHasher
      ph = PasswordHasher()
      hash_contraseña = ph.hash("mi_super_contraseña")
      print("Hash Argon2:", hash_contraseña)
