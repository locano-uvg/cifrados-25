Aplicaciones modernas

¿Criptología?

Campo de estudio que incluye tanto criptografía como criptoanálisis.

¿Criptoanálisis?

Estudio de principios y métodos para descifrar texto cifrado sin conocer la clave.

Desencriptar: Hackers, etc.

¿Criptografía?

Estudio de cómo alterar un mensaje para que no sea legible sin el algoritmo y la clave adecuados.

Encriptar: Científicos

¿Criptografía = Seguridad?

La criptografía es un elemento para cumplir con la seguridad, pero no lo es todo.

Glosario

  • Cifrado: Algoritmo para transformar texto plano en texto cifrado.
  • Texto cifrado: Mensaje codificado.
  • Texto plano: Mensaje original.
  • Criptoanálisis: Métodos para descifrar texto cifrado.
  • Criptografía: Alteración de mensajes para proteger su contenido.
